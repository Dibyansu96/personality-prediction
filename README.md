# personality-prediction
Machine learning project that predicts whether a person is an introvert or extrovert based on text input using Python and natural language processing (NLP).

📌 Overview

This project is a Machine Learning-based Personality Prediction System that predicts whether a person is an Introvert or an Extrovert based on the text they provide. The application uses Natural Language Processing (NLP) techniques to preprocess text and a trained machine learning model to classify personality types.

✨ Features:
Predicts personality from text input.
Classifies users as Introvert or Extrovert.
Text preprocessing using NLP techniques.
Trained machine learning classification model.
Simple and user-friendly interface.
